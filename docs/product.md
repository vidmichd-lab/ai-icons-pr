# Продукт

## Назначение

`AI Icons Studio` — приватный веб-инструмент для пакетной стилизации иконок через `Krea API`.

## Основные сценарии

- загрузка до 10 исходников за один запуск
- текстовый запрос на генерацию иконки без исходного референса
- выбор одного стиля из набора пресетов
- генерация одного варианта на каждый исходник
- повторная генерация через новый `seed`
- скачивание одного `PNG` или `ZIP`, если у сессии несколько готовых результатов

## Поведение интерфейса

- frontend открыт по URL, но рабочий доступ к инструменту идет только после login
- история генераций хранится локально в браузере и изолирована по логину пользователя
- исходные превью в истории сохраняются корректно и остаются видимыми после перезагрузки страницы
- если исходника нет, сессия в истории хранит текстовый запрос и позволяет делать reroll от него
- историю можно вручную очистить
- стили можно добавлять, редактировать и удалять через UI только под `vidmich`
- дефолтные preview пресетов не зависят от локальных frontend-файлов
- без валидной сессии API не дает тратить токены `Krea`
- у каждого пользователя есть лимит 100 генераций в месяц
- оставшийся лимит виден в интерфейсе как `remaining/100`
- у `vidmich` есть встроенная панель управления доступами и генерацией менеджерских логинов

## Ограничения текущей версии

- нет серверного хранения истории
- роли есть только на backend-уровне: `admin` и `manager`
- нет очереди задач вне клиентской сессии
