# Продукт

## Назначение

`AI Icons Studio` — приватный веб-инструмент для пакетной стилизации иконок через `Krea API`.

## Основные сценарии

- загрузка до 10 исходников за один запуск
- выбор одного стиля из набора пресетов
- генерация одного варианта на каждый исходник
- повторная генерация через новый `seed`
- обтравка уже выбранного результата без перегенерации объекта
- скачивание одного `PNG` или `ZIP`, если у сессии несколько готовых результатов

## Поведение интерфейса

- приложение приватное только по URL, без логина
- история генераций хранится локально в браузере
- историю можно вручную очистить
- стили можно добавлять, редактировать и удалять через UI

## Ограничения текущей версии

- нет серверного хранения истории
- нет авторизации и разграничения доступа
- нет пользовательских ролей
- нет очереди задач вне клиентской сессии
