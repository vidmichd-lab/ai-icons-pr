# UI Правила

## Библиотека

Единый UI слой проекта — `shadcn/ui`.

С этого момента:

- новые кнопки делать через `Button`
- новые панели делать через `Card`
- новые модалки делать через `Dialog`
- новые поля делать через `Input` и `Textarea`
- скроллируемые зоны делать через `ScrollArea`
- визуальные маркеры делать через `Badge`
- разделители делать через `Separator`

## Текущий стиль интерфейса

- светлая спокойная палитра
- плотный рабочий layout
- правая колонка приоритетнее, чем управляющие
- минимум декоративного текста
- интерфейс больше похож на рабочий инструмент, чем на лендинг

## Layout

- базовая сетка: `1/4 + 1/4 + 1/2`
- левая колонка: выбор стиля
- средняя колонка: загрузка
- правая колонка: результат и действия

## Принципы

- меньше маркетингового текста, больше операционного UI
- важные действия должны быть очевидны без чтения описаний
- списки стилей и файлов должны быть компактными
- состояние результата важнее декоративности
